# PDF 다운로드 기능 디자인

## 개요

포트폴리오 대시보드 페이지와 공유 포트폴리오 상세 페이지에서 데이터를 PDF로 다운로드하는 기능.
프론트엔드에서 jsPDF + jspdf-autotable을 사용하여 데이터 기반 PDF를 생성한다.
차트는 포함하지 않으며, 수치 데이터와 테이블로만 구성한다.

## 기술 스택

- **jspdf**: PDF 문서 생성
- **jspdf-autotable**: 테이블 자동 생성 플러그인
- **한글 폰트**: Noto Sans KR (public 폴더에 배치, fetch로 로드)

## PDF 콘텐츠 구성

### 1. 포트폴리오 대시보드 PDF

| 섹션 | 내용 |
|------|------|
| 헤더 | 포트폴리오명 (또는 "통합 대시보드") + 생성 날짜 |
| 평가금액 | 현재 평가금액, 전일 대비 변동 |
| 투자 정보 (개별만) | 투자금액, 투자 수익률 |
| 수익률 요약 | 전일대비, 전달대비, 전년대비, YTD (4열 테이블) |
| 리스크 분석 (개별만) | MDD, 변동성, 샤프비율 (1행 테이블) |
| 종목 비중 (통합만) | 종목, 수량, 현재가, 평가금액, 비중 |

### 2. 공유 포트폴리오 PDF

| 섹션 | 내용 |
|------|------|
| 헤더 | 포트폴리오명 + 생성 날짜 |
| 가상 수익률 요약 | 1주, 1개월, 3개월 (3열 테이블) |
| 종목 비중 | 종목, 티커, 비중 + 합계 행 |

## 파일 구조

```
frontend/src/utils/pdfExport.ts       # 공통 유틸 (폰트 로드, 헤더/푸터, 스타일 상수)
frontend/src/utils/dashboardPdf.ts    # 대시보드 PDF 생성 함수
frontend/src/utils/sharedPdf.ts       # 공유 포트폴리오 PDF 생성 함수
```

## UI

- 대시보드 헤더 영역: `Download` 아이콘 버튼 추가
- 공유 포트폴리오 헤더: `Download` 아이콘 버튼 추가
- 클릭 시 즉시 PDF 생성 후 브라우저 다운로드

## 파일명 규칙

- 대시보드: `{포트폴리오명}_대시보드_{날짜}.pdf` 또는 `통합_대시보드_{날짜}.pdf`
- 공유: `{포트폴리오명}_포트폴리오_{날짜}.pdf`
